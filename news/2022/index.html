EPUBTools 0.8.4 -- Conversion to full TypeScript source

 (January 26, 2022) EPUBTools version 0.8.4 has been released, with the primary feature being conversion to TypeScript.

EPUBTools 0.8.3 -- Adopting latest xmldom, TypeScript types

 (January 22, 2022) EPUBTools version 0.8.3 has been released, eliminating a security warning for the xmldom dependency. This release also includes TypeScript type definitions files.
